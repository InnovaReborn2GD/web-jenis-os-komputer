Jenis Sistem Operasi Pada Komputer

Apa itu sistem operasi?

Sistem Operasi adalah sebuah perangkat lunak atau software sistem yang bertugas menjalin hubungan antara hardware dan software komputer dan juga bertugas menjalankan program atau aplikasi komputer. Tanpa sistem operasi, komputer kalian basically cannot functions as it should be. Kalian tidak bisa bermain gim, nonton Netflix, browsing Internet, atau sekedar mengetik di Notepad.

Microsoft Windows

Microsoft Windows adalah sistem operasi yang dikembangkan oleh Microsoft. Windows pertama kali dirilis pada tahun 1985 sebagai Windows 1.0, dan seiring berjalannya waktu, muncullah beberapa versi windows, seperti Windows 9x (95 & 98), Windows 2000, Windows Vista, Windows 7, 8, 10, dan yang paling baru Windows 11, yang dirilis pada Oktober 2021 silam. Windows juga merupakan salah satu sistem operasi terkenal di dunia. Versi populer dari sistem operasi ini adalah Windows 10, diikuti Windows 11 dan Windows 7.

Linux

Linux adalah salah satu sistem operasi Unix-like (yaitu sistem operasi yang mirip dengan Unix). Linux diluncurkan pertama kali pada tahun 1990an oleh Linus Torvalds. Oleh karena sifatnya yang gratis dan open source (kode sumber terbuka), Linux memiliki banyak distribusi, seperti Ubuntu, Kali Linux, Linux Mint, Debian, Fedora, dan lain-lain.

Android

Android adalah sistem operasi yang berbasis Linux. Android diluncurkan pada tahun 2009 sebagai Android 1, kemudian dilanjutkan dengan Android 2, Android 3 Honeycomb, Android 4.x, Android 5.1 Lollipop, Android 6 Marshmallow, dan seterusnya hingga Android 13 Tiramisu, versi yang paling baru. Sebenarnya, Android lebih umum digunakan sebagai sistem operasi pada ponsel, namun kamu bisa menggunakan Android pada komputer dengan menggunakan Android x86.

Chrome OS

Chrome OS adalah sistem operasi berbasis Linux yang dikembangkan oleh Google. Chrome OS didesain sebagai sistem operasi yang difokuskan untuk produktifitas dan juga ringan, karena berbasis web. Meski ringan, penggunaan Chrome OS sangat bergantung pada koneksi internet. Laptop-laptop yang tersertifikasi untuk mengadopsi sistem operasi ini biasanya disebut Chromebook.
